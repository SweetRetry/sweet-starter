# create-sweet

CLI tool to scaffold modern web development starter templates.

## Usage

### npm

```bash
npm create sweet
```

### yarn

```bash
yarn create sweet
```

### pnpm

```bash
pnpm create sweet
```

### npx

```bash
npx create-sweet
```

## Available Templates

| Template | Description | Tech Stack |
|----------|-------------|------------|
| **Next.js Monorepo** | Full-featured monorepo setup | Turborepo, Next.js 16, shadcn/ui, Biome, Knip |
| **React Starter** | Lightweight React SPA | React 19, Vite, Tailwind CSS v4, shadcn/ui |
| **Tauri Starter** | Cross-platform desktop app | Tauri 2, Next.js, Elysia, Turborepo |

## Interactive CLI

The CLI will guide you through:

1. **Template selection** - Choose from 3 starter templates
2. **Project naming** - Enter your desired project name
3. **Auto-setup** - Downloads template and prepares your project

## Prerequisites

- **Node.js** >= 20
- **pnpm** >= 10 (recommended)

Some templates may have additional requirements:
- **Tauri Starter**: Rust, Bun
